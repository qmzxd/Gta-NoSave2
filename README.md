# GTA-NoSave2

Утилита для управления сохранениями в GTA V / Online через `.bat` и `PowerShell`. Позволяет быстро блокировать запись файлов профиля для абуза механик.

### Функционал:
* **Блокировка сохранений**: Переводит файлы профиля в режим "Только чтение" одной кнопкой.
* **Smart Overlay**: Выводит поверх всех окон статус-бар `NO SAVE ENABLED`, чтобы контролировать режим прямо во время игры.
* **Admin Check**: Встроенная проверка прав администратора для корректной работы команд.

### Как использовать:
1. Запустите `NoSave2.bat` **от имени администратора**.
2. Нажмите `1` для включения блокировки (появится белое окно-индикатор).
3. Нажмите `2` для отключения и возврата к обычному режиму.
