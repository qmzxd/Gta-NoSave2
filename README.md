# 🛡️ GTA-NoSave2.1

Утилита для управления сохранениями в GTA V / Online через `.bat` и `PowerShell`. Позволяет быстро блокировать запись файлов профиля для абуза механик (например, финал Кайо-Перико).

### 🚀 Функционал:
* **Блокировка сохранений**: Переводит файлы профиля в режим "Только чтение" одной кнопкой.
* **Smart Overlay**: Выводит поверх всех окон статус-бар `NO SAVE ENABLED`, чтобы контролировать режим прямо во время игры.
* **Admin Check**: Встроенная проверка прав администратора для корректной работы команд.

### 🛠️ Как использовать:
1. Запустите `NoSave2.1.bat` **от имени администратора**.
2. Нажмите `1` для включения блокировки (появится белое окно-индикатор).
3. Нажмите `2` для отключения и возврата к обычному режиму.

---

### 📝 О проекте:
Это мой первый проект. Весь код написан вручную в **Блокноте** при поддержке ИИ для реализации функций PowerShell и оверлея.

**Почему NoSave2?**
Первая версия (`NoSave`) работала нестабильно и не имела визуального индикатора. В версии `NoSave2` я исправил ошибки и добавил **Overlay** — белое окно поверх всех окон, чтобы точно знать, когда режим блокировки активен.

---

### ⚠️ Внимание (Дисклеймер):
Абуз ограблений в GTA Online без перерыва — это нарушение правил игры. Чрезмерное использование скрипта может привести к:
1. Блокировке аккаунта на **1 месяц** с обнулением персонажа.
2. При повторном нарушении — **перманентный бан** (навсегда).
3. Списанию внутриигровой валюты.

*Используйте на свой страх и риск! Автор не несет ответственности за возможные блокировки.*
